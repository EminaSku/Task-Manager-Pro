# Task Manager Pro

Full-stack app (React + Node/Express + PostgreSQL)

## Run locally

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev

## Notes
Create a `.env` file for backend and frontend
